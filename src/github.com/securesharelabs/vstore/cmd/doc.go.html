     7  # Commands
     8  
     9    - `vstore`: Default vStore application startup (ABCI application server).
    10    - `vstore factory`: Create digitally signed transactions for vfs nodes.
    11    - `vstore version`: Print the version number of your vStore instance.
    12    - `vstore info`: Print the current node's vStore information (State).
    13    - `vstore query`: Query your vStore instance for transactions.
    14  
    15  # Examples
    16  
    17  	vstore --home=/tmp/.vfs-home --socket=unix://vfs.sock
    18  	vstore version
    19  	vstore info --home=/tmp/.vfs-home
    20  	vstore factory --home /tmp/.vfs-home --data "Message here" --commit
    21  	vstore query --home /tmp/.vfs-home --hash TRANSACTION_HASH_HEX
    22  */
