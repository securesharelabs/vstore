     1  /*
     2  Package vstore implements a verifiable store for CometBFT nodes.
     3  
     4  vStore is a Go application built using Cosmos SDK. It focusses on providing:
     5  
     6    - data integrity: attaching timestamp and signature information to data ; and
     7    - data redundancy: running a vstore application on top of CometBFT nodes ; and
     8    - data availability: data is available from any supporting CometBFT nodes ; and
     9    - data security: the blockchain is *not* used to store your data!
    10  
    11  vStore is built using cobra: github.com/spf13/cobra
    12  By default, the main function runs the vstoreCmd from `cmd/vstore.go`.
    13  
    14  # Examples
    15  
    16  	vstore --home=/tmp/.vfs-home --socket=unix://vfs.sock
    17  	vstore version
    18  	vstore info --home=/tmp/.vfs-home
    19  	vstore factory --home /tmp/.vfs-home --data "Message here" --commit
    20  	vstore query --home /tmp/.vfs-home --hash TRANSACTION_HASH_HEX
    21  
    22  # Subcommands
    23  
    24    - `vstore`: Default vStore application startup (ABCI server).
    25    - `vstore factory`: Create digitally signed transactions for vfs nodes.
    26    - `vstore version`: Print the version number of your vStore instance.
    27    - `vstore info`: Print the current node's vStore information (State).
    28    - `vstore query`: Query your vStore instance for transactions.
